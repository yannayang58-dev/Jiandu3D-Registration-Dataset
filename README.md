# Jiandu3D-Registration-Dataset

Jiandu3D is a 3D fracture registration dataset for ancient bamboo and wooden slips.

This repository provides dataset documentation, sample data, benchmark splits, and evaluation scripts for the Jiandu3D dataset.

## Dataset

The complete dataset will be released via a public data repository such as Zenodo. This GitHub repository contains documentation, example samples, and code for loading and evaluating the dataset.
